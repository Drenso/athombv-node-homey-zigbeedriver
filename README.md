# homey-zigbeedriver

## Introduction
This module can be used to make the development of Zigbee apps for Homey easier.

It is essentially a map-tool from Homey-capabilities to Zigeee endpoints and clusters.

## Installation

```bash
$ npm install --save homey-zigbeedriver
```
