(static) calculateDimDuration(opts, settings) → {number}

Calculate a transitionTime value for Zigbee clusters, it takes two parameters, opts and settings. The former is the opts object provided by a Homey.Device.registerCapabilityListener which can hold a duration property (in milliseconds), The latter is an object which can hold a transition_time property (in seconds).

(static) convertCIEToHSV() → {HSV}

Method that converts colors from the CIE (xyY) color space to the HSV color space. Note: do not use this for converting xy values from Zigbee devices to HSV, that seems to be inaccurate (see: https://github.com/colorjs/color-space/issues/48).
